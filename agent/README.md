# Agent

This agent runs on PC to receive data via UART from NUCLEO-L476RG board to save data as CSV files on PC.

## File types

```
*.fft ==> FFT
*.raw ==> Raw PCM data
*.flt ==> Hanning window applied on PCM data
```
